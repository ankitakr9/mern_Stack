Introduction of Web-Development

Web development is the process of creating and maintaining websites and web applications that run online, encompassing tasks from designing the user interface to writing the code that makes them functional, secure, and efficient. It involves both front-end development, which focuses on what users see and interact with, and back-end development, which handles the server-side logic, databases, and overall application functionality.

Web-Development Process

The web development process generally involves discovery/analysis, planning, design, development, testing, launch, and maintenance. In the discovery phase, requirements and goals are defined, leading to a plan and site structure. The design phase focuses on the website's visual appearance and user experience. Development involves coding the frontend and backend, followed by thorough testing to fix bugs and ensure functionality. The site is then launched, becoming publicly accessible. Finally, the maintenance phase involves ongoing updates, security checks, and optimization to keep the website running smoothly.

Networking Concepts

What is Internet?

Internet is a global communication system that links together thousands of individual networks. It allows exchange of information between two or more computers on a network. Thus internet helps in transfer of messages through mail, chat, video & audio conference, etc.

What is IP Address?

Internet is a global communication system that links together thousands of individual networks. It allows exchange of information between two or more computers on a network. Thus internet helps in transfer of messages through mail, chat, video & audio conference, etc.

How it Works?

  • Unique Identification: Each device, from computers and smartphones to smart home devices, gets a unique IP address.
  • Location Addressing: The IP address contains location information, which helps route data packets across the network to their final destination.
  • The Internet Protocol (IP): IP is a set of rules for how data packets are formatted and sent across networks, and IP addresses are part of this system.

Types of IP Address-

  • IPv4: The first version of the IP address, a 32-bit number, which became too small to support the growing number of internet devices.
  • IPv6: The next-generation IP address, using 128 bits, provides a vastly larger address space to accommodate the increasing number of connected devices.

How to Access a Website?

When you type a website URL (like www.google.com) in the browser, the browser sends a request to a server. The server finds the website’s data and sends it back to your browser.

Who Assigns the IP Address?

IP addresses are assigned by Internet Service Providers (ISP) and managed globally by IANA (Internet Assigned Numbers Authority).

What is DNS?

DNS (Domain Name System) translates human-readable domain names (like google.com) into IP addresses (like 142.250.190.78).

Problems in Data Transfer

  • Packet loss
  • Network congestion
  • Latency (delay)
  • Security risks (hacking, interception)

MAC Address

A MAC Address is a unique identifier assigned to each device’s network card.

Format: 6 pairs of hexadecimal digits → 00:1A:2B:3C:4D:5E

Port Number

A port number helps identify specific processes/services on a computer. Example: 80 (HTTP), 443 (HTTPS).

Format: Integer value from 0 – 65535

Local Area Network (LAN)

A LAN connects computers within a limited area like a home, school, or office.

Switch & Router

  • Switch: Connects devices in a LAN and forwards data within the network.
  • Router: Connects different networks (e.g., home network to the internet).

Public and Private IP Address

  • Private: Used inside local networks (e.g., 192.168.0.1)
  • Public: Unique IP visible on the internet, assigned by ISP

Modern Routers

Modern routers combine the features of a router, switch, firewall, and Wi-Fi access point in a single device.

What is a VPN?

A VPN (Virtual Private Network) creates a secure, encrypted connection between your device and the internet, hiding your IP and ensuring privacy.
